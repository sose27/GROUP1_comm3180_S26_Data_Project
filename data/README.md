## Data files for COMM3180 Group Project

    
### Data Sources

* American Community Survey (5 Year Estimates)
    * Provides county level data, documenting demographics (race, age), socioeconomics (median household income, unemployment rate, industry, and educational attainment)

* MIT Presidential Election by County
    * Provides county level presidential election results

* California Ballot Measure Data
    * Provides county level voting results for different ballot measures
    * Helps indicate salient issues in counties
